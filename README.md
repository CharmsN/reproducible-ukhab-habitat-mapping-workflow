\# Reproducible UKHab Habitat Mapping Workflow



This repository contains a reproducible workflow developed for an MSc project on UKHab-aligned habitat classification using Sentinel-1, Sentinel-2, LiDAR, and targeted UAV-supported label refinement.



\## Contents

\- notebooks/: main workflow notebook

\- scripts/: supporting processing scripts

\- environment.yml: conda environment for reproducibility



\## Data

Raw data are not included due to file size constraints. Data sources include:

\- Sentinel-1 and Sentinel-2 (Copernicus)

\- UK National LiDAR Programme



\## Usage

Run the workflow notebook step-by-step after setting up the environment.
